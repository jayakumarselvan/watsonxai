
## Create and Activate a Virtual Environment
```
python3 -m venv .venv
source .venv/bin/activate
```

## Install Dependencies
```
pip install -r requirements.txt
```

## .env Configuration
```
WXO_API_BASE_URL = ""
WXO_API_KEY=""
WXO_AGENT_ID_FAQ=""
```

## Run the application:
```
python simple.py
```
